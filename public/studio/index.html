ClaudeBox Studio

  • Add parts from the left. Click a part to select it.
  • Move: drag a selected part on the ground. Raise/lower: Q / E (or the Y field). Rotate: [ / ].
  • Orbit camera: right-drag (or drag empty space). Zoom: wheel.
  • Behaviors (triggers) attach to a part in the inspector: trampoline, launch, kill, checkpoint, finish, teleport, speed, message, spin, move.
  • Save (live) writes the level to the chosen slot. Play tests it right here.
  • The Playground game tile plays the "playground" slot.
